What is the Simple Stock Dashboard?

Frustrated with having to open multiple tabs to see the performance of multiple stocks? Want to get a quick comprehensive overview of your portfolio quickly without being forced to pay for an expensive service?

Simple Stock Dashboard was built with these principles in mind. It aims to provide a holistic overview of your entire portfolio with key metrics, with minimal user input and frustration. Simple Stock Dashboard also collects no information, and the data is only stored in your local computer, under your total control.

Important Info

This website is only meant as a DEMO of the dashboard. The user is supposed to clone the official Github repo and run it locally, so you avoid having to re-upload a CSV everytime you want to track your portfolio performance.

How Do I Use It?

  • The default dashboard displays pre-selected stock information. To track your own, simply upload a CSV file with the columns stockname / boughtprice / boughtamount. Eg.
    • stockname : the symbol of the stock
    • boughtprice : the price you bought the stock (optional)
    • boughtamount : the quantity of stock you purchased at boughtprice
  • An example can be found here: Input Example
  • The dashboard generates a summary for all your stock at the top of the page, and individual graphs + stock metrics for each stock that you specify.
